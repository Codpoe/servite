Servite

用于 React SSR / SSG / CSR / Islands 的 Vite 插件

✨ SSR

插件默认开启 SSR，但也支持 SSG 和 CSR。

⚡️ SSG

如果站点大部分内容都是静态的，可方便地切换到 SSG。

🖥 CSR

当服务端渲染不满足需求时，也可以切换到 CSR 纯浏览器端渲染。

🏝 Islands

支持 Islands 孤岛架构，比 SSG 更进一步，性能更好。

💡 Nitro

Servite 底层由 Nitro 驱动。借助 Nitro 可实现 API 接口和中间件。
