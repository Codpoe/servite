# 目录结构

默认情况下，servite 对目录结构有一些约定：

```shell
<root>
└─ src                # 源码目录
   ├─ pages           # 页面路由目录
   │  └─ ...
   └─ server          # 服务端源码目录
      ├─ middlewares  # 中间件目录
      │  └─ ...
      └─ routes       # 服务端路由目录
         └─ ...
```

可以通过 [source](/zh/guide/config#source) 配置项修改这些约定。
